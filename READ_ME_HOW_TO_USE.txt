HOW TO WORK ON THIS PROJECT ON ANY COMPUTER:
--------------------------------------------------------------
1. Plug this pen drive into your laptop or another computer.
2. If you want to view the website locally, just double-click:
   ➡️ START_WEBSITE.bat
   (Note: You will need Node.js installed on the computer to run it locally)

3. TO MAKE CHANGES USING GOOGLE ANTIGRAVITY / CLAUDE:
   - Provide the AI with the folder path to this project on your pendrive.
   - For example: "My website project is located at G:\xavier school website. Please open it!"
   - The AI will read the files directly from the pen drive and make changes here.

4. TO PUBLISH CHANGES TO THE INTERNET:
   - Make sure your local changes are working.
   - Upload all the updated files from this folder directly to your GitHub repository:
     https://github.com/susmanjha9-collab/stxaviers-school-website
   - Render.com will detect the GitHub updates and automatically restart/republish your website online!
